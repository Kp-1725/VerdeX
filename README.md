# Hackathon Project

Team Name: JOBIESS

## Problem Statement

Farm-to-market supply chains often suffer from low transparency, delayed status updates, pricing uncertainty, and poor trust between farmers, retailers, and end consumers.

## Solution

VerdeX is a role-based agri traceability platform that combines blockchain-backed journey proofs, QR-based public tracking, and a live operations dashboard.

Key capabilities:

- Product creation and lifecycle tracking from harvest to market
- Role-based workflows for farmers and retailers
- Trade request inbox with status updates and messaging
- Public product journey page via QR link
- Price transparency with farmer and retail pricing
- Government price recommendation flow with fallback support
- ML-based short-term price forecast integration
- Multilingual interface support

## Tech Stack

- Frontend: React, Vite, Tailwind CSS
- Backend: Node.js, Express
- Database: MongoDB
- Blockchain: Solidity, Ethers.js, MetaMask, Ganache
- ML Service: Python Random Forest API

## How to Run

1. Install dependencies

- Backend: go to backend folder and run npm install
- Frontend: go to frontend folder and run npm install

2. Configure environment files

- Copy backend/.env.example to backend/.env
- Copy frontend/.env.example to frontend/.env

3.Launch the start-app.bat file

- Desktop: http://localhost:5173
- Mobile on same Wi-Fi: http://172.16.7.78:5173
